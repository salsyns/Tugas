1. Apa gunanya IFRAME?

IFRAME (Inline Frame) adalah elemen HTML yang digunakan untuk menyisipkan dokumen HTML atau konten web lainnya ke dalam halaman web yang sedang ditampilkan. IFRAME memungkinkan Anda untuk menampilkan konten dari sumber eksternal seperti website lain, video, peta, atau sumber daya web lainnya secara terenkapsulasi di dalam halaman web sendiri

2. Apa bedanya FRAME dan IFRAME?

- FRAME: FRAME adalah elemen HTML yang digunakan untuk membagi satu halaman web menjadi beberapa jendela atau "frame" yang dapat menampilkan dokumen HTML yang berbeda. FRAME digunakan dalam elemen untuk membuat tata letak berbasis frame dalam satu dokumen HTML.
- IFRAME: IFRAME adalah elemen HTML yang digunakan untuk menyisipkan satu dokumen HTML atau sumber daya web eksternal ke dalam halaman web utama. IFRAME tidak membagi halaman web menjadi frame terpisah seperti FRAME, tetapi menampilkan konten tambahan di dalam satu elemen berbeda di halaman utama.
