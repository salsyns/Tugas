1. Apa gunanya CSS?

CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mengontrol tampilan dan tata letak elemen-elemen HTML pada sebuah halaman web. Fungsi utama CSS adalah untuk memisahkan struktur (HTML) dan gaya (styling) dari suatu halaman web, sehingga memungkinkan pengembang web untuk mengubah tampilan secara konsisten di seluruh situs web dengan mengedit satu set peraturan gaya (style rules).
